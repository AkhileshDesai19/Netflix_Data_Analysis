# Netflix Data Analysis (EDA) Project

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a Netflix dataset using Python. The objective is to analyze the dataset, uncover trends, identify patterns, and generate meaningful insights through data visualization.

The project demonstrates key data analytics skills such as data cleaning, data exploration, visualization, and insight generation.

---

## Dataset

The dataset contains information about movies and TV shows available on Netflix, including details such as:

* Title
* Genre
* Release Year
* Vote Average
* Vote Count
* Popularity
* Original Language
* Adult Content Flag

## Project Workflow

### 1. Data Loading

* Imported the dataset into Pandas DataFrame.
* Examined the structure and dimensions of the dataset.

### 2. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Corrected data types where necessary.
* Handled inconsistent values.

### 3. Exploratory Data Analysis (EDA)

* Analyzed movie popularity trends.
* Examined vote averages and vote counts.
* Explored content distribution across genres.
* Investigated release year patterns.
* Identified highly rated and popular titles.

### 4. Data Visualization

Created visualizations using Matplotlib and Seaborn to better understand:

* Distribution of ratings
* Popularity trends
* Genre-wise analysis
* Release year distribution
* Top-performing content

---

## Key Insights

* Netflix contains content from multiple genres and languages.
* Certain genres dominate the platform's catalog.
* Highly popular titles tend to receive higher engagement from viewers.
* Content production has increased significantly in recent years.
* Ratings and popularity metrics provide valuable insights into audience preferences.
